# novim

A friendly terminal editor for vibe coders. Not vim.

## Install

```bash
curl -fsSL novim.dev/install | bash
```

Or via Homebrew:

```bash
brew install novim
```

## Usage

```bash
novim              # Open editor
novim file.txt     # Open file
```

Press `?` for help.

## Shortcuts

| Key | Action |
|-----|--------|
| Ctrl+S | Save |
| Ctrl+Z | Undo |
| Ctrl+Shift+Z | Redo |
| Ctrl+A | Select all |
| Ctrl+C | Copy |
| Ctrl+V | Paste |
| Esc Esc | Quit |
| ? | Help |

## Features

- Mouse support (click, drag, select)
- Standard keyboard shortcuts
- File tree on the left
- No vim knowledge required

## Requirements

- Neovim (installed automatically via Homebrew)
