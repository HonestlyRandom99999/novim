novim

A friendly terminal editor for vibe coders. Not vim.

curl -fsSL novim.dev/install | bash

VSCode-like

Just type to edit. No modes to learn.

Standard Shortcuts

Ctrl+S, Ctrl+Z, Ctrl+C/V. No learning curve.

Mouse Support

Click, drag, select. Double-click to open.

File Tree

Built-in file browser on the left.

Dynamic Hints

Status bar shows relevant shortcuts.

Safe Quit

Prompts to save before closing.
